# Customer Behavior Analysis

## Overview

This project analyzes customer purchasing behavior using Python, SQL, and Power BI to uncover valuable business insights. The workflow includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis in PostgreSQL, interactive dashboard creation, report writing, and presentation development. The project demonstrates an end-to-end data analytics pipeline suitable for real-world business decision-making.

---

## Dataset

The dataset contains customer transaction and product-related information, including customer details, purchase history, product categories, ratings, discounts, subscription status, and other behavioral attributes.

## Project Workflow

### 1. Data Loading

* Imported the dataset using Python.
* Performed initial data inspection.

### 2. Exploratory Data Analysis (EDA)

* Analyzed data distributions and relationships.
* Identified missing values, duplicates, and outliers.
* Created visualizations to understand customer behavior.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicates.
* Corrected data types.
* Prepared the dataset for analysis.

### 4. SQL Analysis

* Imported the cleaned dataset into PostgreSQL.
* Wrote SQL queries to answer business questions.
* Used aggregate functions, joins, window functions, and grouping techniques.

### 5. Power BI Dashboard

* Built an interactive dashboard.
* Added KPIs, charts, slicers, and filters.
* Visualized customer behavior and sales trends.

### 6. Report & Presentation

* Prepared a detailed project report.
* Created a professional presentation using GAMMA AI.

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Run the Python notebooks/scripts for data cleaning and EDA.
4. Import the cleaned dataset into PostgreSQL and execute the SQL queries.
5. Open the Power BI (.pbix) file to explore the dashboard.
6. Review the project report and presentation for detailed findings.
